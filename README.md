# Tecnologias utilizadas
- HTML / CSS / TypeScript
- Angular 13
- Angular Material
# Como executar o projeto

Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/carlosjeff/pessoa-crud.git
# instalar dependências
npm install
# executar o projeto
npm run start
```
